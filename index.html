EL PODER DEL HTML

1.- HISTORIA Y ELEMENTOS DEL HTML

I) 1990 - CREACIÓN DEL HTML

  • Tim Berners-Lee diseña el concepto de World Wide Web.
  • Crea la primera versión de HTML con unas 18 etiquetas básicas.

II) 1995 - HTML 2.0

  • Primer estándar oficial por IETF
  • Incorpora formularios y tablas simples para interacción con el usuario.

III) 1997 - HTML 3.2

  • HTML 3.2 lanzado por W3C
  • Añade compatibilidad con JavaScript y applets para contenido dinámico.

IV) 2014 - HTML5

  • Soporte nativo para audio, video, gráficos y nuevos elementos semánticos.
  • Nuevos elementos semánticos (header, footer, article).

V) 2019 - HTML Living Standard

  • WHATWG y W3C acuerdan mantener HTML como estándar vivo (Living Standard).
  • Mantenimiento continuo por WHATWG.

2.- ELEMENTOS DEL HTML

I) ESTRUCTURA Y METADATOS: Configuran y describen el documento; no se muestran directamente en el contenido visible.

  • Ejemplo: <html> head <meta>, <title>

II) ESTRUCTURA DE CONTENIDO: Organizan el contenido en bloques semánticos que definen la estructura de la página.

  • Ejemplo: <body>, <header>, <nav>, <main>

III) TEXTOS Y TIPOGRAFIA: Muestran y formatean el texto y los encabezados.

  • Ejemplo: <h1>, <p>, <br>, <strong>

IV) LISTAS: Presentan conjuntos de ítems organizados de forma ordenada, desordenada o de definición.

  • Ejemplo: <ul>, <ol>, <li>, <dl>

V) ENLACES Y NAVEGACION: Conectan el documento con otros recursos internos o externos.

  • Ejemplo: <a>, <link>

VI) MULTIMEDIA: Insertan imágenes, audio, video u otros contenidos externos en la página.

  • Ejemplo: <img>, <video>, <audio>, <source>

VII) TABLAS:Organizan datos en filas y columnas.

  • Ejemplo: <table>, <tr>, <td>, <th>

VIII) FORMULARIOS: Recogen datos o interacciones del usuario para procesarlos.

  • Ejemplo: <form>, <input>, <textarea>, <label>

IX) INTERACTIVOS Y SCRIPTING: Ofrecen funcionalidades dinámicas y controles para el usuario.

  • Ejemplo: <details>, <summary>, <dialog>, <template>
